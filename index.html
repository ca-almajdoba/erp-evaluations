2. إذا كنت تعدل على ملف CSS منفصل (.css): أضف سطر الاستيراد في أول سطر من الملف تماماً، ثم حدد العناصر التي تريد تغيير خطها: CSS /* 1. استيراد الخط في أعلى الملف */ @import url('https://fonts.googleapis.com/css2?family=Open+Sans:wght@400;700&display=swap'); /* 2. تعيين الخط للعناصر */ body, p, h1, h2, h3, span { font-family: 'Google Sans', sans-serif; } ملاحظة هامة: خط "Google Sans" الأصلي هو خط محمي بحقوق ملكية لشركة Google وغالباً ما يكون متاحاً للاستخدام العام في مكتبة الخطوط باسم "Open Sans" أو "Product Sans". الأكواد أعلاه تستدعي البديل الرسمي المتاح والممتاز للمواقع. إذا كنت تريدني أن أعدل لك الكود بدقة عالية، انسخ الكود الخاص بك واكتبه هنا في ردك القادم وسأقوم بتعديله لك فوراً!
